## About

Solid Pancakes is an event-based consumer that tracks usages on the TFS products.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
